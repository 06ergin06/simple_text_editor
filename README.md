# Hakki Text Editor
This is a simple text editor.

### Techs
- Python3
- GTK4

### AppImage Build
To build the application as an AppImage:
```bash
./build-appimage.sh
```

The AppImage will be created as `Hakki-Text-Editor-x86_64.AppImage` and can be run with:
```bash
./Hakki-Text-Editor-x86_64.AppImage
```

For detailed build instructions, see [APPIMAGE_BUILD.md](APPIMAGE_BUILD.md)
